How to Use CYNQ Workflow

What is CYNQ?

CYNQ is a visual workflow tool based on the 5 stages of Human-Centered Design (Design Thinking). It helps you organize ideas, research, and prototypes as you develop solutions to user problems.

The 5 Stages:

1. üíô Empathize - Understand your users through research, interviews, and observation.
Example: "User struggles with finding parking downtown"
2. üéØ Define - Frame the problem clearly based on your research.
Example: "How might we help commuters find parking quickly?"
3. üí° Ideate - Brainstorm creative solutions without judgment.
Example: "Real-time parking app with availability maps"
4. üîß Prototype - Build quick, low-cost versions to test your ideas.
Example: "Wireframe mockup of app interface"
5. üß™ Test - Gather feedback and refine your solution.
Example: "5 users found the map confusing - simplify UI"

How to Use:

  1. Create Cards: Click "+ Add Research/Problem/Idea" in any stage to create a card
  2. Drag & Drop: Move cards between stages as your project progresses
  3. Edit/Delete: Use the pencil and trash icons on each card
  4. Iterate: Move back and forth between stages - it's not always linear!
üí° Pro Tip: Start with Empathize - good design begins with understanding real user needs!
